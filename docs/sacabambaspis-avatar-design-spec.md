## 1. 形象来源与参考

### 1.1 迷因来源

萨卡班甲鱼（学名 *Sacabambaspis janvieri*，日语 サカバンバスピス）是已灭绝的奥陶纪无颌甲胄鱼（约 4.7 亿年前）。迷因形象来自**芬兰赫尔辛基自然历史博物馆**的硅胶复原模型，由爱沙尼亚古生物学家 Elga Mark-Kurik 于 1995 年制作。眼睛是从工艺品商店购买的**娃娃眼**，外观粗糙呆萌，这恰恰成为其爆火原因。

### 1.2 参考来源

| 来源 | 内容 | 可靠性 |
|------|------|--------|
| [Wikipedia - Sacabambaspis](https://en.wikipedia.org/wiki/Sacabambaspis) | 古生物学描述、身体结构、尾巴解剖学 | 高（学术来源） |
| [KnowYourMeme - Sacabambaspis](https://knowyourmeme.com/memes/sacabambaspis) | 迷因起源、传播历史、二次创作 | 高（迷因档案） |
| 赫尔辛基自然历史博物馆实物模型 | 原始迷因形象 | 最高（原始素材） |

### 1.3 关键澄清

- 萨卡班甲鱼**不是翻车鱼**（Mola mola），两者分类、外形、时代完全不同
- "サカバンバンガ"是中文互联网对日语名称的变体音译，标准日语标签是 `#サカバンバスピス`

## 2. 外观规格

### 2.1 整体形状

蝌蚪状（tadpole-like），分为两大区域：

| 区域 | 特征 |
|------|------|
| **头部甲胄区** | 扁平盾牌形骨质甲片，上方平坦后部微隆，下方前部深凹。表面有**橡树叶形状的瘤状突起**（tubercles）。头部两侧一排弯曲的**菱形鳃盖板**。尺寸约占总长 30%。 |
| **躯干鳞片区** | 头部之后急剧收窄，覆盖窄小鳞片，锥形逐渐变细到尾部。 |

总长度比例：头部甲胄占约 30%，躯干+尾部占约 70%。

### 2.2 从前向后截面半径变化

| 区域 | s 参数值 | 横截面形状 | 宽度（归一化） | 说明 |
|------|---------|-----------|--------------|------|
| 嘴部前端 | s=0 | 窄椭圆 | 0.15 | 无颌，始终张开的窄缝 |
| 眼部区域 | s≈0.08 | 宽扁椭圆 | **1.0（最宽）** | 两眼间距极近，位于最前端 |
| 头部中段 | s≈0.15 | 宽扁椭圆 | 0.95 | 甲胄最宽处 |
| 头部后缘 | s≈0.25 | 宽扁椭圆→急剧收窄 | 0.7 | 鳃盖板过渡区 |
| 甲胄-躯干过渡 | s≈0.30 | 椭圆 | 0.35 | **急剧收窄**，宽度骤降 |
| 躯干前段 | s≈0.40 | 小椭圆 | 0.20 | |
| 躯干中段 | s≈0.55 | 更小椭圆 | 0.12 | |
| 躯干后段 | s≈0.70 | 细椭圆 | 0.07 | |
| 尾柄 | s≈0.85 | 极细圆 | 0.03 | |
| 尾尖 | s=1.0 | 极细 | 0.01 | |

**核心比例**：头部宽度约为躯干最大宽度的 **3-4 倍**。从头甲后缘到尾部的过渡非常急剧。

### 2.3 各部位详细规格

#### 眼睛

- 两眼间距**极近**，位于头部**最前端**
- 类似汽车前灯的排列
- 圆圆的黑色娃娃眼，带**白色高光**
- 眼睛之间只有一个很小的 T 形骨板
- 表情始终是"目瞪口呆"地凝视前方（像颜文字 `⊙▽⊙`）
- 眼睛大小：约占头部宽度的 25-30%

#### 嘴巴

- **无颌**（jawless）
- 始终张开的窄缝，呈倒三角形或椭圆形
- 下"唇"由一束薄鳞片条构成
- 嘴和眼睛之间有一对小鼻孔，紧贴每只眼睛旁边
- **嘴巴不能闭合**——这是区别于当前实现的关键特征

#### 鳃盖

- 头部两侧一排弯曲的**菱形鳃盖板**
- 从上方看呈弧形排列

#### 鳍

- **除了尾巴之外没有任何鳍**
- 没有胸鳍、背鳍、腹鳍——这是甲胄鱼类的典型特征
- 当前代码中的背鳍/腹鳍必须移除

#### 尾巴

- **鞭状/鼠尾状**（whip-like / rat-tail）
- 极细的鳞片管包裹脊髓
- 尾鳍膜分**上下两部分**（上歪尾/hypocercal）
- 脊髓在尾部向下弯曲，大部分鳍膜在脊髓上方
- 尾鳍整体呈**细长的柳叶形**
- 尾巴可以有轻微摆动，但幅度很小

#### 颜色

| 元素 | 颜色 | 说明 |
|------|------|------|
| 甲胄上半部 | `#a09888`（灰棕） | 带瘤状突起纹理 |
| 甲胄下半部 | `#d8d0c0`（米白） | 鱼肚白 |
| 躯干鳞片区上半 | `#908878`（深灰棕） | 鳞片纹理 |
| 躯干鳞片区下半 | `#c8c0b0`（浅米） | |
| 眼白 | `#ffffff` | |
| 瞳孔 | `#1a1a1a` | 带白色高光点 |
| 嘴巴内部 | `#3a2a2a`（暗色） | 始终可见的窄缝 |
| 鳃盖板 | `#7a7060`（深灰棕） | 菱形，略深于甲胄 |
| 尾鳍 | `#8a8070`（中灰棕） | 半透明质感 |

## 4. 柔韧度物理规格

### 4.1 设计目标

让萨卡班甲鱼动起来时看起来像一条**软软的鱼**：
- 头部甲胄区**僵硬**（骨质甲片不可弯曲）
- 躯干鳞片区**柔软**（可以侧向弯曲）
- 抬头/低头时身体缓慢跟上，有**惯性延迟**
- 身体与头部保持**有机整体**（不是脱节的两段）

### 4.2 物理模型

```
头部姿态（来自面捕）→ 立即应用
    ↓
躯干姿态 → 通过弹簧-阻尼系统延迟跟随头部
    ↓
尾巴姿态 → 进一步延迟跟随躯干
```

#### 弹簧-阻尼参数建议

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 躯干跟随刚度 (stiffness) | 3.0 - 5.0 | 越大越快跟上头部 |
| 躯干阻尼 (damping) | 0.7 - 0.9 | 越大振荡越少 |
| 尾巴跟随刚度 | 1.5 - 3.0 | 比躯干更慢 |
| 尾巴阻尼 | 0.6 - 0.8 | |
| 最大弯曲角度 | 15° - 25° | 躯干相对头部的最大偏转 |

#### 实现方式

在 `ProceduralMeshRenderer` 的 `_update()` 循环中：

1. 接收面捕参数（yaw/pitch/roll）
2. 头部立即应用这些角度
3. 躯干各截面（s > 0.25）的角度通过弹簧插值延迟跟随：
   - `bodyAngle[s] = lerp(bodyAngle[s], headAngle, stiffness * dt)`
   - 加入阻尼：`bodyAngle[s] *= (1 - damping * dt)`
4. 尾巴截面（s > 0.7）进一步延迟
5. 渲染时，每个截面的旋转角度 = 头部角度 * followFactor(s)，其中 followFactor 从 1.0（头部）平滑衰减到约 0.3（尾尖）

#### followFactor 曲线建议

```
s ∈ [0, 0.25]  → followFactor = 1.0  （头部甲胄，刚体）
s ∈ [0.25, 0.5] → followFactor = 1.0 → 0.7 （过渡区，开始柔软）
s ∈ [0.5, 0.75] → followFactor = 0.7 → 0.4 （躯干中段）
s ∈ [0.75, 1.0] → followFactor = 0.4 → 0.2 （尾巴，最柔软）
```

用 smoothstep 或指数衰减实现平滑过渡。

### 4.3 侧向弯曲

除了跟随头部旋转外，躯干还应有自然的**侧向摆动**：

- 基础摆动：`sin(time * frequency) * amplitude * (1 - s)`，越靠近尾尖幅度越大
- 频率：约 0.5-1.0 Hz（慢悠悠的摆动）
- 幅度：约 3-5°（微妙的，不是剧烈的）
- 面捕驱动时：摆动叠加在面捕姿态之上

## 5. mesh 顶点数据规格

### 5.1 截面形状函数

头部甲胄区（s ∈ [0, 0.25]）应为**宽扁椭圆**：

```
rx(s) = headWidth * radiusScale(s)    // 左右宽度，较大
ry(s) = headHeight * radiusScale(s)   // 上下高度，较小
// headWidth / headHeight ≈ 1.8 - 2.2（宽扁比）
```

躯干区（s ∈ [0.25, 1.0]）急剧收窄，截面趋近圆形再变细：

```
rx(s) = bodyWidth * tailScale(s)
ry(s) = bodyHeight * tailScale(s)
// bodyWidth / bodyHeight ≈ 1.2 - 1.5（略扁）
```

### 5.2 radiusScale 曲线

```
s ∈ [0, 0.08]     → 快速增长到 1.0（眼部区域最宽）
s ∈ [0.08, 0.25]  → 缓慢下降到 0.7（甲胄后缘）
s ∈ [0.25, 0.35]  → 急剧下降到 0.25（甲胄-躯干过渡）
s ∈ [0.35, 0.70]  → 持续收窄 0.25 → 0.08
s ∈ [0.70, 1.0]   → 缓慢收尾 0.08 → 0.01
```

### 5.3 建议的参数化默认值

```javascript
const SACA_DEFAULTS = {
  headWidth: 80,       // 甲胄左右半径（宽）
  headHeight: 40,      // 甲胄上下半径（扁）
  headDepth: 35,       // 甲胄前后深度
  bodyWidth: 18,       // 躯干最大左右半径
  bodyHeight: 14,      // 躯干最大上下半径
  bodyLength: 200,      // 躯干总长度（从甲胄后缘到尾尖）
  tailFlukeHeight: 12,  // 尾鳍上下展开高度
  tailFlukeLength: 25,  // 尾鳍前后长度
  columns: 36,          // 沿主轴的截面数
  rows: 24,             // 每个截面的顶点数
};
```

## 6. 面部特征绘制规格

### 6.1 眼睛

| 属性 | 规格 |
|------|------|
| 位置 | 头部最前端（s ≈ 0.05），两眼间距 ≤ 眼睛直径的 30% |
| 大小 | 约占头部宽度的 25-30% |
| 形状 | 正圆形 |
| 眼白 | 纯白 `#ffffff` |
| 瞳孔 | 黑色 `#1a1a1a`，占眼睛直径的 50-60% |
| 高光 | 白色小圆点，位于瞳孔右上方，直径约瞳孔的 20% |
| 眨眼 | 可以眨眼（上下眼睑从上方关闭），但默认状态是完全睁开的"目瞪口呆" |
| 旋转行为 | 随头部刚体旋转，不独立移动 |

### 6.2 嘴巴

| 属性 | 规格 |
|------|------|
| 位置 | 眼睛正下方偏后（s ≈ 0.08） |
| 形状 | **始终张开的窄缝**，倒三角形或椭圆形 |
| 宽度 | 约两眼间距的 60-80% |
| 高度（张开度） | 固定小开口（约 3-5 单位），不随面捕 mouthOpen 参数闭合 |
| 颜色 | 内部暗色 `#3a2a2a` |
| 面捕映射 | mouthOpen 参数可以微调开口大小（0.5x - 1.5x），但**不会完全闭合** |
| 微笑 | mouthSmile 参数可以让嘴角微微上翘 |

### 6.3 鼻孔

| 属性 | 规格 |
|------|------|
| 位置 | 紧贴每只眼睛旁边（略偏下偏外） |
| 大小 | 很小，直径约 2-3 单位 |
| 形状 | 小椭圆 |
| 颜色 | 暗色 `#2a2a2a` |

### 6.4 鳃盖板

| 属性 | 规格 |
|------|------|
| 位置 | 头部两侧（s ≈ 0.15 - 0.25） |
| 形状 | 弯曲的菱形，3-5 片排列成弧形 |
| 颜色 | 比甲胄略深 `#7a7060` |
| 绘制方式 | 可以用简单的菱形多边形 + 描边实现 |

### 6.5 瘤状突起

| 属性 | 规格 |
|------|------|
| 位置 | 甲胄表面（s ∈ [0.05, 0.25]） |
| 形状 | 橡树叶形（椭圆形 + 尖端） |
| 大小 | 小突起，直径约 3-6 单位 |
| 分布 | 半随机分布，密度约每 10x10 单位面积 1-2 个 |
| 绘制方式 | 用比甲胄色略深/略浅的小椭圆模拟凸起质感 |

## 7. 尾巴规格

### 7.1 形状

鞭状/鼠尾状，不是鲸鱼尾叶。

```
躯干末端（s≈0.85）→ 极细尾柄（直径约 2-3 单位）
    → 继续延伸（s≈0.95）→ 脊髓向下弯曲
    → 尾尖（s=1.0）→ 上歪尾鳍膜（柳叶形）
```

### 7.2 尾鳍膜

| 属性 | 规格 |
|------|------|
| 形状 | 柳叶形/细长叶片形 |
| 上鳍膜 | 较大，从尾柄上方向后展开 |
| 下鳍膜 | 较小或不明显 |
| 颜色 | `#8a8070`，半透明质感 |
| 摆动 | 可随身体姿态轻微摆动，幅度 5-10° |

### 7.3 与当前实现的区别

当前 `mesh-spindle-whale.js` 中的尾巴是**鲸鱼型分叉尾叶**（上下两片对称扇形），需要改为**鞭状上歪尾**（一条细长尾巴 + 尾尖处的小型柳叶形鳍膜）。
